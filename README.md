Full-stack web app that helps users discover movies, TV shows, and anime with AI-driven recommendations.

- **Stack:** Python · Flask · Supabase (PostgreSQL) · TMDB API · Groq API · Vercel

### [AI Browser Extension](https://github.com/tirth6851/ai-browser-extension) — *Claude-Controlled Browser Automation*
Chrome extension that intercepts structured `[ACTION: ...]` tags from Claude.ai and executes real browser actions.

- **Stack:** JavaScript · Chrome Extension APIs (content scripts + service worker)
- Supports Supabase actions (create tables, retrieve API keys, manage projects) and Vercel actions (list deployments, check logs, manage env vars)
- Confirmation toast before any action executes — no credential storage
- Planned expansion to GitHub, Railway, PlanetScale/Neon, ChatGPT, and Gemini
